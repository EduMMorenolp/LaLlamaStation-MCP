# Gestión de Changelog

Cualquier modificación en el código debe ser registrada en `CHANGELOG.md`.

## Instrucciones para la IA:
- **Momento**: Actualizar el changelog inmediatamente después de cada cambio significativo.
- **Formato**: Usar el estándar "Keep a Changelog".
- **Categorías**: Añadido, Mejorado, Corregido, Cambiado, Eliminado.
- **Idioma**: Español.

No concluir ninguna tarea sin haber actualizado el historial de cambios.
